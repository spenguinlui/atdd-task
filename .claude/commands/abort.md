# Abort Task

## 檢查 Active 任務

呼叫 `atdd_task_list()` 取得所有任務，過濾出 status 不是 `completed`、`aborted`、`verified` 的 active 任務。

**如果沒有 active 任務：**
```markdown
┌──────────────────────────────────────────────────────┐
│ ℹ️ 沒有進行中的任務可以放棄                          │
└──────────────────────────────────────────────────────┘
```

---

## 確認放棄

**如果有 active 任務，先顯示確認：**

```markdown
┌──────────────────────────────────────────────────────┐
│ ⚠️ 確認放棄任務                                      │
├──────────────────────────────────────────────────────┤
│                                                      │
│ 🏷️ 類型：{type}                                     │
│ 📁 專案：{project}                                   │
│ 📝 標題：{description}                               │
│ 📍 當前階段：{status}                                │
│                                                      │
│ ═══ 已完成的工作 ═══                                 │
│ {列出已完成的階段和產出}                             │
│                                                      │
│ ⚠️ 放棄後：                                          │
│ • 任務將移至 failed/ 目錄                            │
│ • 已產生的檔案不會自動刪除                          │
│ • 可以稍後查看任務記錄                              │
│                                                      │
│ 確定要放棄嗎？輸入 "yes" 確認，或任意鍵取消         │
└──────────────────────────────────────────────────────┘
```

---

## 執行放棄

如果用戶確認（輸入 "yes"）：

### Step 1: 更新任務狀態

執行 `shared/task-state-update.md` 的 **`task-cancelled`** 事件：

- reason = "用戶手動放棄"

> 此事件會統一處理：Task JSON 更新（status=failed）、檔案移動至 failed/、Kanban 放棄。

### Step 2: 輸出確認

```markdown
┌──────────────────────────────────────────────────────┐
│ 📋 任務已放棄                                        │
├──────────────────────────────────────────────────────┤
│                                                      │
│ 🏷️ 類型：{type}                                     │
│ 📝 標題：{description}                               │
│ 📍 放棄於：{status} 階段                             │
│                                                      │
│ 📁 記錄已保存至 DB                                   │
│                                                      │
│ ═══ 清理建議 ═══                                     │
│ 如果需要刪除已生成的檔案，請手動處理：              │
│ {列出可能需要刪除的檔案}                            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

---

