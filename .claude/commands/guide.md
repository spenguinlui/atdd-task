## 快速開始

如果你是第一次使用，建議：

1. **開始一個新功能**
   ```
   /feature sf_project, 你想做的功能名稱
   ```

2. **回答 specist 的問題**，直到信心度達到 95%

3. **輸入 `/continue`** 進入下一階段

4. **重複步驟 3** 直到任務完成

## 常見問題

**Q: 如何查看當前有哪些任務？**
A: 輸入 `/status` 或 `/atdd-list`

**Q: 可以同時進行多個任務嗎？**
A: 可以。`/continue` 會列出所有 active 任務讓你選擇，或用 `/continue {project}` 指定

**Q: 信心度是什麼？**
A: specist 評估需求清晰程度的指標，所有任務類型需達 95%
